# Anthropic Deep Competency Program

**US Cohort 1 | New York City | June 15-26, 2026**

A two-week immersive training program building deep Claude competency across 6 practitioner personas.

## Structure

```
curriculum/
  week-1/          # In-person NYC (Jun 15-19)
    day-1.md       # Foundations + Claude Code
    day-2.md       # API Deep Dive + Prompt Engineering
    day-3.md       # Tool Use + Agents
    day-4.md       # MCP + Extended Thinking
    day-5.md       # Assessment + Build Kickoff
  week-2/          # Virtual (Jun 22-26)
    day-6.md       # AI Product Process
    day-7.md       # Build Sprint 1
    day-8.md       # Build Sprint 2
    day-9.md       # Build Sprint 3
    day-10.md      # Team Demos + Close
```

## Program Details

- **Participants:** 12 intermediate-to-advanced practitioners
- **Personas:** Data Engineer, AI Engineer, Architect, Software Engineer, Business Analyst/Consultant, Product Manager
- **Facilitators:** 3 (PPMD, Sr. Manager, Manager)
- **Tracks:** Competency Track (5 Claude domains) + Build Track (AI product process)
- **Format:** Week 1 in-person (NYC), Week 2 virtual
