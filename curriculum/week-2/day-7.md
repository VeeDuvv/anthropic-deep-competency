## Masterclass: Claude Code Productivity Patterns (45 min)

- The "plan then build" workflow: `/plan` → review → approve → implement
- Multi-file refactoring: how to give Claude enough context
- Test-driven development: write tests first, let Claude implement
- Code review with Claude: using it as a reviewer, not just a writer
- Custom slash commands for repetitive tasks
- When to restart vs. continue a conversation (context window management)

## Masterclass: Testing + Evaluation for AI Apps (45 min)

- The testing pyramid for AI: unit tests → integration tests → eval suites → human review
- Deterministic tests: input/output pairs, schema validation, error handling
- LLM-as-judge evaluation: using Claude to evaluate Claude
- Regression testing: golden datasets, drift detection
- A/B testing prompts: systematic prompt improvement
- Cost tracking: monitoring API spend during development
