---

## Masterclass: Production Deployment Patterns (45 min)

- API key management: environment variables, secrets managers, rotation
- Rate limiting and retry logic: exponential backoff, circuit breakers
- Cost controls: budget alerts, model routing (Haiku for simple, Opus for complex)
- Monitoring: latency tracking, error rates, token usage dashboards
- Scaling: batching, caching, async processing
- Security: input sanitization, output filtering, audit logging

## Masterclass: Writing Technical Comms (45 min)

- Why internal comms matter: adoption doesn't happen without awareness
- The internal case study format:
  - Problem (1 paragraph)
  - Approach (what Claude features, what architecture)
  - Results (quantified where possible)
  - Lessons learned (what surprised you, what you'd do differently)
  - Reusable patterns (what others can copy)
- Writing for different audiences: engineers vs. managers vs. executives

## Demo Prep Workshop (30 min)

- Demo structure: Problem → Live Demo → Architecture → Lessons → Q&A
- Tips: rehearse, have a backup recording, keep it under 15 minutes
- What judges look for: real problem, clean execution, Claude features used, team collaboration
